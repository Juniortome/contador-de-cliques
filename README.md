🚀 Contador React
Este é um projeto simples de contador desenvolvido para praticar os fundamentos do React, como a criação de componentes, manipulação de estados com o Hook useState e manipulação de eventos.

📋 Sobre o Projeto
A aplicação permite que o usuário incremente um valor numérico através de um clique e resete esse valor para zero quando desejar. É um exercício clássico para entender a reatividade do React.

🛠️ Tecnologias Utilizadas
React (Biblioteca JavaScript)

Vite (Ferramenta de build rápida)

JavaScript (ES6+)

CSS3 (Estilização inline e básica)

🚀 Como executar o projeto
Para rodar este projeto localmente, siga os passos abaixo:

Clone o repositório:

Bash
git clone https://github.com/Juniortome/contador-de-cliques
Entre na pasta do projeto:

Bash
cd nome-do-projeto
Instale as dependências:

Bash
npm install
Inicie o servidor de desenvolvimento:

Bash
npm run dev
Acesse no navegador:
O Vite geralmente disponibiliza o endereço http://localhost:5173.

🧠 O que eu aprendi
Hooks (useState): Como gerenciar o estado de um componente de forma funcional.

Event Handling: Como capturar cliques do usuário e disparar funções.

Renderização Condicional/Dinâmica: Como o React atualiza o DOM automaticamente quando o estado muda.
